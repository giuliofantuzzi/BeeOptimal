Installation
============

Installing via PIP
------------------

.. code-block:: bash

   pip install beeoptimal


Installing from source
----------------------

1. Clone the repository:
   
   .. code-block:: bash

      https://github.com/giuliofantuzzi/BeeOptimal.git

2. Move into the repository directory and install the package with:
   
   .. code-block:: bash

      cd BeeOptimal/
      pip install .
